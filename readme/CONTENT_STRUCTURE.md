# Content Structure Guide
## Supreme Animation Studio Website

This document clearly outlines what content is needed for each section of the website.

---

## 📁 SECTION 1: PORTFOLIO / SELECTED WORKS

### **Section Purpose:**
Showcase completed projects and demonstrate capabilities through real work examples.

### **What This Section Shows:**
- **Past work** - Actual projects you've completed
- **Visual proof** - Images/videos of your work
- **Results** - What you've created for clients
- **Categories** - Types of work you do

### **Content Needed:**

#### **Section Header:**
- **Portfolio Section Description** (1-2 sentences)
  - Brief description of your portfolio/work philosophy
  - Example: "Exploring the boundaries of digital artistry and cinematic storytelling"

#### **For Each Project (4 projects needed):**

1. **Project Title**
   - Name of the project
   - Example: "Neon Genesis", "Cyber Void"

2. **Project Category**
   - Type of work
   - Example: "3D Animation", "Motion Graphics", "VFX", "Art Direction"

3. **Project Description**
   - Brief 1-2 sentence description
   - What the project is about, what was created

4. **Client Name** (Optional)
   - Company or client name (if you can share)

5. **Project Year**
   - When the project was completed
   - Example: "2024", "2023"

6. **Project Thumbnail**
   - Image or video preview
   - Recommended: 1200x1200px image or short video
   - This will be displayed on the project card

7. **Case Study Link** (Optional)
   - URL to detailed project page or case study

### **Where Content Appears:**
- Project cards in the grid layout
- Hover effects show additional info (client, year)
- "View Project" button (can link to case study)

---

## 📁 SECTION 2: SERVICES / OUR EXPERTISE

### **Section Purpose:**
Describe what services you offer, your capabilities, and how you work with clients.

### **What This Section Shows:**
- **What you offer** - Services available to clients
- **Capabilities** - What you can do
- **Deliverables** - What clients receive
- **Industries** - Who you work with

### **Content Needed:**

#### **Section Header:**
- **Services Section Description** (1-2 sentences)
  - Brief description of your services/expertise
  - Example: "Comprehensive animation and visual effects services for brands and storytellers"

#### **For Each Service (4 services needed):**

1. **Service Title**
   - Name of the service
   - Example: "3D Animation", "Motion Graphics", "VFX & Simulation"

2. **Service Description**
   - Brief 1-2 sentence description
   - What this service includes, what you do
   - Example: "Bringing characters and worlds to life through cutting-edge 3D animation techniques"

3. **Deliverables**
   - What clients receive when they hire you
   - Example: "Character animations, scene renders, final video files, project files"

4. **Industries Served**
   - Which industries or client types you serve
   - Example: "Entertainment, Gaming, Advertising, Film & TV"

5. **Process Overview** (Optional)
   - Brief description of your process/approach
   - Example: "Concept development → Storyboarding → 3D Modeling → Animation → Rendering → Post-production"

### **Where Content Appears:**
- Service list items (large, interactive)
- Description appears on hover (desktop)
- Clean, minimal design with red accent on hover

## 💡 TIPS FOR CLIENT

- **Be specific** - Detailed descriptions work better than vague ones
- **Use examples** - Reference similar work if helpful
- **Think visually** - Consider what images/videos best represent each project
- **Be consistent** - Use similar tone and style across all content
- **Quality over quantity** - Better to have 4 strong projects than 8 weak ones
