# Studio Pigeon Design Analysis

## Key Features That Make Their Hero "Vibrant & Classy"

### 1. **Hero Section Elements**
- **Clean Typography**: Large, bold headline with excellent spacing
- **Prominent Showreel**: "Showreel 2024" featured prominently with video player
- **Social Integration**: Instagram link directly in hero ("Get regular, moving updates")
- **Clear Value Prop**: "Meet Pigeon Studio" with compelling description
- **Clean Layout**: Not cluttered, excellent use of white space
- **Professional Tone**: Professional yet approachable

### 2. **Portfolio Showcase**
- Large project cards with hover effects
- Client names, years, team members visible
- Categories clearly labeled
- Social sharing options
- Case study links

### 3. **Services Section**
- Clear categorization
- Industry-specific offerings
- Detailed descriptions

### 4. **Overall Design Principles**
- **Minimal but Impactful**: Clean design with strategic use of color
- **Video-First**: Showreel prominently featured
- **Social Proof**: Client testimonials and logos
- **Clear CTAs**: Multiple engagement points
- **Professional Credibility**: Team info, years of experience
